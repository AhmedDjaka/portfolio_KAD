## 🚀 Comment Utiliser

### 1. Ouverture du Portfolio
```bash
# Simplement ouvrez index.html dans un navigateur
# Ou utilisez un serveur local:
python -m http.server 8000
# Puis accédez à http://localhost:8000
```

### 2. Personnalisation
- **Informations personnelles**: Modifiez les données dans `index.html`
- **Couleurs**: Changez les variables CSS dans `styles.css` (`:root`)
- **Contenu**: Mettez à jour les sections texte et les images
- **Projets**: Ajoutez vos propres projets dans la section "Projets"

### 3. Déploiement
- Uploadez les fichiers sur votre serveur web
- Supporté par GitHub Pages, Netlify, Vercel, etc.
- Aucune base de données requise (site statique)
